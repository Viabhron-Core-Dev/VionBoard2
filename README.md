# VionBoard

**VionBoard** is a secure, private, and lightweight keyboard for Android, built by [Viabhron](https://github.com/elschuyler/VionBoard).

Based on [HeliBoard](https://github.com/Helium314/HeliBoard) — a privacy-conscious open-source keyboard.

## Themes
- 🔒 **Secure** — two-tier vault auth, overlay protection, double encryption
- 🕵️ **Private** — no internet, no telemetry, no analytics, local crash logs only
- ⚙️ **Stable** — crash recovery, integrity checks, graceful fallbacks
- 🎨 **Customisable** — themes, toolbar, shortcuts, dictionary import
- 🪶 **Lightweight** — dormant features, memory ceiling enforced

## Features
- Improved suggestion strip — bigram engine, contextual numbers
- Personal shortcuts expander — multi-word expansion
- Integrated vault (KeePass .kdbx) — fills password fields directly
- Magic keyboard panel — biometric/master password unlock
- Voice recording + optional offline Vosk transcription
- Extended toolbar with quick-lock, clipboard clear, gesture and multilingual toggles
- Offline dictionary system with manual import
- Separate keyboard and vault backup systems

## Status
Active development — personal/hobby project.
